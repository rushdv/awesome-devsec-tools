# Contributing to Dev & Security Toolkit

Thank you for your interest in contributing! 🎉

## How to Add a Tool

1. **Fork** this repository
2. **Find the right category** in `README.md`
3. **Add your tool** using this format:

```markdown
| **Tool Name** | [source.com](https://source.com) | One-line description | Platform/Language |
```

## Guidelines

- Tools must be **actively maintained** (last commit within 2 years)
- Include the **official website or GitHub link**
- Keep descriptions **concise** (under 10 words)
- No paid tools unless they have a meaningful free tier
- Cybersecurity tools must be for **ethical/defensive** use only

## Submitting

1. Create a new branch: `git checkout -b add-toolname`
2. Make your changes
3. Submit a Pull Request with title: `Add: [Tool Name] to [Category]`

Thank you for helping the community! 🙏
